# Python-CUGAN-AI-Upscaler
 - Uses CUGAN SuperRes AI to Upscale &amp; Enhance Videos, Gifs, and Images

Run the "Requirements" file to install Python Requirements

This also needs FFMPEG
FFMPEG: https://www.ffmpeg.org
FFMPEG Install Guide: https://youtu.be/JR36oH35Fgg

This also needs Python (3.10.6 will work best)
Python can be found here: https://www.python.org
Python to PATH Install guide: https://youtu.be/H7iqfv5dzRY

Put your media (Videos, Gifs or Images) in the "MEDIA" folder
Run the "Start" file
Pick a resolution
Pick a worker count (recommended to use 1 on weaker systems, if your mouse lags while moving it when running this, lower worker count)
Let the script run
Your upscaled media is in the "OUTPUT" folder
The original file will be moved from MEDIA to FINISHED







This uses an AI model called CUGAN, this is much faster than ESRGAN, but slightly less quality, and works a lot better on weaker systems
